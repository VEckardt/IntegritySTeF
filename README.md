# IntegritySTeF
Setup Testing Framework for PTC Integrity

It uses Fitnesse, a Java Wiki for the test execution.

##  Development environment
- PTC Integrity LM 10.9 (also 11.0 should be fine)
- Netbeans 7.4 (or 8)
- Java 1.7 (or 1.8)
- Fitnesse framework
